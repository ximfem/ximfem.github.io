注册ChatGPT账号最新完整流程

发表于2023-08-23|更新于2023-08-23|技术分享原创
|字数总计:955|阅读时长:2分钟|阅读量:|评论数:

前言

ChatGPT依然是目前较火的大语言模型。ChatGPT在国内的镜像站已经成千上万，之前郎哥也有写过镜像站的搭建教程（点此回顾），然而镜像站是通过API key来对接ChatGPT服务，每一次提问都是需要收费的，但如果直接在ChatGPT官网提问的话就是免费（不过官网对网络环境要求比较苛刻，如果没有一定的技术，建议还是放弃），所以这时候注册一个ChatGPT官网账号就能体验到白嫖聊天的乐趣。关于注册ChatGPT账号的文章，网上也有很多，但很多内容都过时了，所以郎哥写下这篇教程，教大家用最新的方式注册ChatGPT账号。

注册账号

注册账号之前首先要确保你已经科学上网了，并且节点最好是美国的，亚洲的基本都不行

打开ChatGPT官网，点击“Sign up”

输入邮箱(现在不能用QQ邮箱了，最好用谷歌邮箱)，或者使用下面三个快捷授权登录，点击“Continue”

设置一个容易记得住的密码，点击“Continue”

需要验证邮箱

登录邮箱，会看到一封验证的邮件，点击绿色的按钮打开验证链接

完成验证后会让你填写信息（姓名和出生日期），随便填就行，然后点击“Continue”

到这里需要验证手机号（国内的手机号不行，所以需要去接码平台租国外的手机号来验证）

接码

目前用得最多的就是俄罗斯的接码平台SMS-Activate.org（网址就是这个）

点击右上角的“登录注册”

注册账号

填写信息，点击注册

完成人机验证后再点一次注册

需要验证邮箱，在邮箱里找到这封邮件，点击“确认”即可

确认之后就可以登录账号了。登录后点击右上角的+号进行充值

充值方式选择支付宝

最低充值2美元，大概15块钱左右，点击“支付”

支付宝扫码

完成支付

搜索openai，点击第一个结果

选择便宜一点的地区号码

复制号码（如果二十分钟内没有收到短信，可以随时点击右边的X号取消租用，费用会全部返还）

选择对应的国家，填入刚刚的号码，点击“Send code”

收到验证码，将验证码填入框内即可

注册成功

登录ChatGPT

点击右上角的头像→Visit ChatGPT

点击“Log in”

输入账号和密码，点击“Continue”

登录成功

可以开始聊天了

查看API key

每个新注册的账号都会赠送5美元的API额度(有效期一般是四个月)，通过API key可以将自己的服务接入OpenAI，或者在ChatGPT镜像站直接使用ChatGPT，无需科学上网。

点击右上角的头像→View API key

在Usage页面可以看到额度的使用情况和有效期等信息

进入API key页面，创建一个key

自定义名称，然后点击创建

创建完成后赶紧复制下来保存好，因为这个密钥之后显示一次，后面就没法查看了。如果你没有保存，只能重新创建一个新的密钥

注意

1、全程不要随便换节点，不然很容易被封号。

2、API key自己保存好，不要泄露，不然就相当于把自己的账号让给别人了

3、使用接码平台接码时如果当前手机号收不到验证码就换一个手机号，不会扣费的
