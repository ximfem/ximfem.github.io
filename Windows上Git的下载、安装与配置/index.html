Git是目前非常流行的分布式版本控制工具，也是开发者必不可少的的工具之一。下面郎哥就教大家在Windows上下载、安装和配置Git。

下载

方法一

直接去官网下载最新版本，但是官网的下载速度真的很慢

方法二

下载郎哥提供的安装包（获取方式在文末）

安装

运行下载好的安装包，点击Next

选择安装目录（尽量选择C盘以外的地方），Next

···

其他步骤全部默认即可

···

点击Install进行安装

取消勾选View Release Notes，点击Finish完成安装

查看右键菜单，可以发现多了两个功能

在右键菜单中点击Git Bash Here

这就打开了Git的命令框
